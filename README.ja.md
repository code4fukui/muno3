# muno3


JavaScriptで書かれた、シンプルなルールベースの対話型チャットボットです。古典的な和歌を詠んだり、計算を行ったり、地元の鯖江の観光に関する質問に答えたり、富山の飲食店情報を検索したりすることができます。

## デモ

**[muno3を試す (人工無能 ver 3.1)](https://code4fukui.github.io/muno3/)**

## 特徴

-   **対話型の応答:** 一般的な挨拶（時間帯に対応）、別れの挨拶、「あなたの名前は？」「何ができるの？」といった簡単な質問に対応します。
-   **百人一首:** 100首の古典的な和歌からランダムに1首を詠んだり、最初の数文字に基づいて和歌を検索したりします。
-   **日付と時刻:** 現在の時刻、曜日、今年の干支に関する質問に答えます。
-   **計算:** 九九の練習、割勘の計算、簡単な数式の評価を行うことができます。
-   **シンプルなナレッジベース:** 内部辞書を使用して、場所、概念、文化的な事実に関する事前に定義された質問（例：「福井県の人口は？」、「七草とは？」）に答えます。
-   **地域情報:** 鯖江市の観光スポット情報を提供し、`opendata-toyama`ライブラリを使用して富山県の飲食店を検索することができます。

## 使い方

1.  このリポジトリをクローンします。
2.  Webブラウザで `index.html` を開きます。
3.  入力ボックスにメッセージを入力し、「発言」を押します。

**質問の例:**
-   `こんにちは` (こんにちは)
-   `詠んで` または `百人一首` (和歌を詠む)
-   `今何時？` (今何時ですか？)
-   `福井県の名物は？` (福井県の名産品は何ですか？)
-   `富山のラーメン` (富山のラーメン)

## 依存関係

このプロジェクトはブラウザ上で動作し、いくつかのJavaScriptモジュールと外部データソースに依存しています。

-   **JSモジュール:**
    -   [CSV.js](https://code4fukui.github.io/CSV/CSV.js)
    -   [rnd.js](https://js.sabae.cc/rnd.js)
    -   [Num.js](https://js.sabae.cc/Num.js)
-   **データソースライブラリ:**
    -   [富山県オープンデータ](https://github.com/code4fukui/opendata-toyama/) の [EateryToyama.js](https://code4fukui.github.io/opendata-toyama/EateryToyama.js)
