# muno3

人工無能 ver 3.1による鯖江の観光案内が体験できるシステムです。

## 機能

- 時間応答
- 歌唱
- 百人一首
- 割勘計算
- 富山県オープンデータからの情報検索

## 必要環境

特になし。ブラウザで動作します。

## 使い方

ブラウザで[デモページ](https://code4fukui.github.io/muno3/)を開き、会話を行ってください。

## データ・API

- [富山県オープンデータ](https://github.com/code4fukui/opendata-toyama/)
